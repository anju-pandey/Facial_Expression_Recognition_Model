Introduction:
Our goal is to build a model which can classify different human facial expressions. Provided dataset contains four types of expressions: 'happy', 'sad', 'neutral' and 'surprised'.

In Milestone-1 and Milestone-2, investigative and comparative study was done to decide on the best solution for this problem statement. The outcome of the study suggested below:

  • Using grayscale images
  • Building model with CNN
  • Developing complex neural network architecture, instead of using pretrained classification model(s)

    Using above findings, we will build a CNN model. Data exploratory analysis was also performed in previous part hence we will omit that section here. Let’s begin with model architecting.
